Memoria Estatica ^OUlhlU1I

El tamaño de la memoria estatica permanece fijo durante la ejecucion del programa.
La memoria estatica se divide en areas (segmentos):
    - codigo
    - datos

Se accede a los datos estaticos por sus direccinoes de memoria absolutas. Siendo los datos
comunmente incluidos en la memoria estatica los siguientes:
    - Variables globales
    - Variables estaticas y de clase
    - Identificadores constantes
    - Literales como cadenas y raramente numeros reales

Los datos estaticos tienen direcciones de memoria crecientes, comenzando en cero. ^HjHmRPwm

(3) IntType: type → INT ^PwDqTJVv

(4) CharType: type → CHAR ^ErEsqAFj

(5) RealType: type → DOUBLE ^iJHiS2RO

(7) ArrayType: type1 → type2 INT_CONSTANT ^QjqBYvED

(8) RecordType: type → field* ^3ZkAJVNL

(9) RecordField: field → ID type ^PF8mi8jM
