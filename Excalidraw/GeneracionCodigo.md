# Text Elements
Simbolos y Tipos ^t08W4HWE

Tanto los simbolos (variables) como los tipos son usados en la generacion de codigo:
    
    - Los simbolos (variables) se utilizan para conocer las direcciones de memoria de las variables
                         ( direccion base + desplazamiento/offset)

    - Los tipos incorporan el tamañana y la representacion de cada variable

 ^bBxrCTjb

TIPOS ^Fa8jGJF7

MAPL tiene una palabra de 2 bytes:
    - Char 1byte
    - Int 2bytes
    - Double 4bytes
    - memDir 2bytes

ARRAYS
    Para implementar el metodo numberOfBytes habra que realizar lo siguiente:

         @Override
          public int getNumberOfBytes() {
            return type.getNumberOfBytes() * size;
          }


REGISTROS (STRUCT)
    El tamaño de un struct es la suma de los tamaños de los tipos de los campos.
    El desplazamiento de memoria relativo de cada campo es la suma de los tamaños de los tipos de
    los campos anteriores.

         ^Nzbws38M
